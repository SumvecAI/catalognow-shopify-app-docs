# Events & Labels

## Events Calendar

Create events to automatically label products for sales, holidays, and promotions.

### Event Types

- **Sale Event** — flash sales, seasonal sales, clearance
- **Holiday** — Christmas, Diwali, Black Friday
- **Festival** — music festivals, sports events, cultural events

### Creating an Event

1. Go to **Events**
2. Click **Create Event**
3. Fill in: name, type, start/end dates
4. Optionally set label prefix and text
5. Toggle auto-apply on/off
6. Click **Create**

### Auto-Apply Labels

When enabled, matching products automatically receive labels during the event period. Labels are removed when the event ends.

## Product Labels

Labels appear on product detail pages:
- **EVENT** — applied by an active event
- **MANUAL** — applied manually
- **AI_SUGGESTED** — suggested by AI agents

Labels can have expiration dates and are automatically cleaned up.
