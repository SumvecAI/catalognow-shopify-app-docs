CatalogNow — AI Product Catalog Enrichment for Shopify • Last updated: March 22, 2026

1. Service Description

CatalogNow is a Shopify application that provides AI-powered product catalog enrichment. The service includes:

  • AI-generated product descriptions, categories, and SEO metadata
  • Vision-based product image analysis
  • Content quality scoring and recommendations
  • Batch product enrichment operations
  • Two-way Shopify product synchronization

2. Account Requirements

To use CatalogNow, you must:

  • Have an active Shopify store
  • Install the app through the Shopify App Store or development install
  • Provide a valid OpenRouter API key for AI features (free tier available)
  • Grant the required Shopify API scopes (read_products, write_products, read_metaobjects)

3. API Key Responsibility

CatalogNow requires an OpenRouter API key to access AI models. You are responsible for:

  • Obtaining your own OpenRouter API key at openrouter.ai

CatalogNow does not charge for AI model usage — costs are between you and OpenRouter. Free models are available that require no credits.
